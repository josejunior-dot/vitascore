# Política de Privacidade — VitaScore

**Última atualização:** Abril 2026

## 1. Sobre o VitaScore

O VitaScore é um aplicativo de monitoramento de hábitos de vida que ajuda usuários a acompanhar atividade física, sono, nutrição, peso e bem-estar emocional.

## 2. Dados coletados

O VitaScore coleta os seguintes dados, **armazenados exclusivamente no dispositivo do usuário**:

- **Atividade física:** passos, distância, calorias (via Health Connect / HealthKit)
- **Sono:** horário de dormir e acordar, duração
- **Nutrição:** fotos de refeições (armazenadas localmente), registros alimentares
- **Peso:** registros de peso e IMC
- **Bem-estar digital:** tempo de uso do aplicativo
- **Bem-estar emocional:** respostas ao check-in WHO-5 (voluntário e anônimo)
- **Localização:** GPS durante monitoramento de treino (somente quando ativado pelo usuário)

## 3. Armazenamento e segurança

- **Todos os dados pessoais ficam armazenados EXCLUSIVAMENTE no dispositivo do usuário.**
- Nenhum dado pessoal de saúde é enviado para servidores externos sem consentimento explícito.
- Os dados são protegidos por criptografia do dispositivo.
- Cada registro possui hash SHA-256 para garantir integridade.

## 4. Compartilhamento de dados

- O VitaScore **NÃO compartilha** dados pessoais com terceiros.
- Quando utilizado em ambiente corporativo, a empresa recebe **apenas dados agregados** (médias, percentuais), nunca dados individuais.
- Nenhum dado individual de saúde é visível para empregadores, seguradoras ou terceiros.

## 5. Health Connect / Apple HealthKit

- O VitaScore pode ler dados de saúde do Google Health Connect ou Apple HealthKit.
- A leitura é feita localmente no dispositivo.
- O usuário autoriza explicitamente quais dados são acessados.
- O VitaScore não escreve dados no Health Connect / HealthKit.

## 6. Fotos

- Fotos de refeições e balança são armazenadas localmente no dispositivo.
- As fotos podem ser analisadas por IA para avaliação nutricional, mediante consentimento.
- As fotos não são enviadas para servidores sem ação explícita do usuário.

## 7. Direitos do usuário (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), o usuário tem direito a:

- **Acesso:** visualizar todos os seus dados a qualquer momento no aplicativo
- **Portabilidade:** exportar todos os dados em formato JSON
- **Eliminação:** apagar todos os dados com um toque (Perfil → Configurações → Apagar dados)
- **Revogação:** desativar qualquer tipo de coleta a qualquer momento
- **Informação:** saber exatamente quais dados são coletados e como são usados

## 8. Dados de menores

O VitaScore não é destinado a menores de 18 anos. Não coletamos intencionalmente dados de menores.

## 9. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data da última atualização está indicada no topo do documento.
